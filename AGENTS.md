# AGENTS.md

## Project Type
This is a Unity psychological horror game.

## Main Idea
The player lives alone in a 1990s house. During the day, they complete simple tasks. At night, they enter surreal dream spaces. Actions in dreams change the real world.

## Important Files
Before coding, read:
- HorrorGame/00_START_HERE.md
- HorrorGame/Current_Task.md
- HorrorGame/Progress.md
- HorrorGame/Bugs.md

## Coding Style
- Use Unity C#.
- Keep scripts beginner-friendly.
- Add comments.
- Use simple names.
- Avoid complicated architecture unless asked.
- Do not delete existing work unless necessary.

## Unity Folder Structure
Put scripts here:
- Assets/Scripts

Put scenes here:
- Assets/Scenes

Put prefabs here:
- Assets/Prefabs

Put materials here:
- Assets/Materials

## After Making Changes
Update:
- HorrorGame/Progress.md

If something breaks, write it in:
- HorrorGame/Bugs.md
